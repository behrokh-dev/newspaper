# 📰 Newspaper Layout Demo

A simple **newspaper-style layout** built using pure **CSS Columns**.  
This project demonstrates how to create multi-column text layouts similar to traditional print newspapers — clean, readable, and responsive.

## ✨ Features

- Multi-column layout using `column-count` and `column-gap`
- Clean and minimal design inspired by classic newspapers
- Responsive behavior for different screen sizes
- Pure HTML & CSS (no frameworks)

---

## 🛠️ Built With

- HTML5  
- CSS3 (Columns Module)

---

## 📚 What I Practiced

- Using CSS **multi-column layout**
- Controlling text flow between columns
- Managing spacing and readability
- Creating print-style UI on the web
